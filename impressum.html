Wir sind Mascha, Lena, Klara und Joris und haben uns im Rahmen von Jugendhackt überlegt, wie wir die Welt mit Code verbessern könnten. Unser Denkansatz war, dass Schüler in der Schule fast nichts mitbestimmen können, bzw meist nicht einmal Bescheid wissen. Um das zu ändern haben wir eine Website erstellt, auf der nach dem Wiki-Prinzip User Beiträge schreiben können und eben diskutieren können.

Den gesamten Code gibt es auf Github zum nachlesen.
